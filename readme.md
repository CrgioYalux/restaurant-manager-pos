### Requisitos de Desarrollo
- Usar Yarn como manejador de paquetes. Para instalarlo:
```bash
$ npm install --global yarn
```
- Una vez instalado Yarn, instalar las dependencias del proyecto ejecutando:
```bash
$ yarn install
```
